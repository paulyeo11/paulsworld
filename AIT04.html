Why Install Claude Code? (And Is It Worth It?) 为什么要安装Claude Code？（值得吗？）

Before we talk about how to install Claude Code, let me explain why you might want to bother. After all, Claude already works perfectly in a web browser — why go through installation? 在我们讨论如何安装Claude Code之前，让我解释一下为什么你可能想费心去做这件事。毕竟，Claude已经在网页浏览器中运行得很好了——为什么要经历安装过程呢？

The answer is power and control. Claude Code — when installed on your computer — can directly read and edit files on your machine, run commands, manage your projects, and make changes that automatically update your website. It is not just a chatbot. It is more like hiring a highly skilled assistant who can sit at your computer and do the technical work for you while you watch. 答案是能力和控制。安装在电脑上的Claude Code可以直接读取和编辑你机器上的文件、运行命令、管理你的项目，并做出自动更新你网站的更改。它不仅仅是一个聊天机器人，更像是雇用了一个高技能助手，可以坐在你的电脑前替你完成技术工作，而你只需在旁观看。

💡 What Claude Code (Desktop) Can Do That Browser Claude Cannot 💡 桌面版Claude Code能做而浏览器版Claude做不到的事
  • Read and edit files directly on your computer — no copy-pasting needed直接读取和编辑你电脑上的文件——无需复制粘贴
  • Run terminal commands on your behalf (install packages, start servers, etc.)代表你运行终端命令（安装软件包、启动服务器等）
  • Work on your entire project folder at once — not just one file at a time同时处理整个项目文件夹——而不是一次只处理一个文件
  • Push changes to GitHub automatically from your own machine从你自己的机器自动推送更改到GitHub
  • Understand your full codebase context — even large projects with hundreds of files理解你的完整代码库上下文——即使是有数百个文件的大型项目

The first time Claude Code read all my files, understood my website's style, and created a new article — pushing it live without me typing a single line of code — I felt like I had unlocked a superpower.当Claude Code第一次读取我所有的文件，理解我网站的风格，并创建了一篇新文章——在我没有输入一行代码的情况下就将其推送上线——我感觉自己解锁了一项超能力。

Before You Start — What You Need 开始之前——你需要什么

Installation requires just two things. Do not be put off by the names — I will explain each one in plain language. 安装只需要两样东西。不要被这些名称吓到——我会用通俗语言解释每一个。

✅ What You Need Before Installing ✅ 安装前你需要的东西
  • Node.js (version 18 or higher) — This is a free piece of software that allows your computer to run JavaScript programs outside of a browser. Claude Code is built on it. Think of it like a required "engine" that Claude Code runs on. Download it free from nodejs.org. Node.js（18版本或更高）——这是一个免费软件，允许你的电脑在浏览器之外运行JavaScript程序。Claude Code就是建立在它之上的。把它想象成Claude Code运行所需的"引擎"。可从nodejs.org免费下载。
  • An Anthropic Account with API Access — You need an account at anthropic.com and an API key. This is what connects Claude Code on your computer to Anthropic's AI brain. There is a small cost for API usage — but for personal use it is very affordable (pennies per conversation). 具有API访问权限的Anthropic账户——你需要在anthropic.com注册一个账户并获取API密钥。这就是将你电脑上的Claude Code连接到Anthropic AI大脑的东西。API使用有少量费用——但对于个人使用来说非常实惠（每次对话只需几分钱）。

Step-by-Step: Installing Claude Code 一步步：安装Claude Code

Follow these steps in order. Take your time — there is no rush. Each step is simpler than it looks. 按顺序遵循这些步骤。慢慢来——不用着急。每一步都比看起来简单。

Install Node.js on Your Computer在你的电脑上安装Node.js

Go to nodejs.org in your browser. Click the big green button that says "LTS" (this means Long-Term Support — the stable, recommended version). Download the installer and run it like any normal software installation. Click Next, Next, Install, Finish. That is all.在浏览器中打开nodejs.org。点击显示"LTS"的大绿色按钮（这意味着长期支持——稳定的推荐版本）。下载安装程序并像安装任何普通软件一样运行它。点击下一步、下一步、安装、完成。就这些。

To confirm it installed correctly, open Terminal and type:要确认安装正确，打开终端并输入： $ node --version
v22.0.0   # You should see a number like this你应该看到这样的数字
Open Your Terminal (the Black Screen)打开你的终端（黑色屏幕）

On Mac: press Command + Space, type "Terminal", press Enter. On Windows: press the Windows key, type "Command Prompt" or "PowerShell", press Enter. A window with a blinking cursor will appear. This is the terminal — do not be afraid of it. You only need to type a few short commands.在Mac上：按Command + 空格，输入"Terminal"，按回车。在Windows上：按Windows键，输入"Command Prompt"或"PowerShell"，按回车。会出现一个带有闪烁光标的窗口。这就是终端——不要害怕它。你只需要输入几个简短的命令。

Install Claude Code with One Command用一个命令安装Claude Code

In the terminal, type the command below exactly and press Enter. This downloads and installs Claude Code globally on your computer. The -g means "install it everywhere so it works from any folder."在终端中，准确输入下面的命令并按回车。这会在你的电脑上全局下载和安装Claude Code。-g意味着"到处安装它，以便它可以从任何文件夹工作。"

Paste this into your terminal and press Enter:将此粘贴到你的终端并按回车： $ npm install -g @anthropic-ai/claude-code

You will see text scrolling past as it downloads. This usually takes 1–2 minutes. When it stops and shows the cursor again, it is done.你会看到文字滚动显示下载过程。通常需要1-2分钟。当它停止并再次显示光标时，就完成了。

Navigate to Your Project Folder导航到你的项目文件夹

Claude Code works best when you open it from inside your project folder — the folder where your website files live. Use the cd command to navigate there. cd stands for "change directory" — it is how you move around folders in the terminal.Claude Code在你从项目文件夹内部打开时效果最好——即存放你网站文件的文件夹。使用cd命令导航到那里。cd代表"change directory"（更改目录）——这是在终端中浏览文件夹的方式。

Example — replace with YOUR folder path:示例——用你的文件夹路径替换： $ cd ~/Documents/my-website

Tip: On a Mac, you can drag a folder from Finder directly into the terminal window after typing cd — it will fill in the path automatically!提示：在Mac上，输入cd 后，你可以将文件夹从Finder直接拖到终端窗口——它会自动填写路径！

Launch Claude Code for the First Time第一次启动Claude Code

Type claude and press Enter. On your very first run, Claude will ask for your Anthropic API key. This is a long string of letters and numbers you get from your Anthropic account dashboard.输入claude并按回车。在你第一次运行时，Claude会要求你提供Anthropic API密钥。这是一串从你的Anthropic账户控制台获得的长字母和数字。

Start Claude Code:启动Claude Code： $ claude

Paste your API key here and press Enter. Claude will confirm it works and show you a prompt — you are ready to start!在这里粘贴你的API密钥并按回车。Claude会确认它可以工作并显示提示——你已经准备好开始了！

Getting Your Anthropic API Key 获取你的Anthropic API密钥

If you do not have an API key yet, here is how to get one. This is the part that confused me at first — but it is straightforward once you know where to look. 如果你还没有API密钥，这是获取方法。这是最初让我困惑的部分——但一旦你知道去哪里找，就很简单了。

Create an Anthropic Account创建Anthropic账户

Go to console.anthropic.com and sign up. Use the same email you use for Claude chat if you already have one. Verify your email when prompted.访问console.anthropic.com并注册。如果你已经有Claude聊天账户，使用相同的电子邮件。在提示时验证你的电子邮件。

Add Credit to Your Account向你的账户添加余额

Click on "Billing" and add a small amount — even $5 is more than enough to start with. API usage is priced per token (pieces of text), so personal projects typically cost just a few cents per day.点击"Billing"并添加少量金额——即使是5美元也足以开始。API使用按token（文本片段）计费，因此个人项目通常每天只需花费几分钱。

Create Your API Key创建你的API密钥

In the Anthropic console, click "API Keys" in the left sidebar, then click "Create Key". Give it a name like "My Computer" so you remember what it is for. Copy the key — it looks like sk-ant-api03-...在Anthropic控制台中，点击左侧边栏中的"API Keys"，然后点击"Create Key"。给它起一个名字，比如"我的电脑"，这样你就记得它是用来做什么的。复制密钥——它看起来像sk-ant-api03-...

🔐

Keep your API key secret. Never paste it into a file you share online or post it publicly. Treat it like a password. If you ever think it has been exposed, go back to the Anthropic console and delete it, then create a new one. 保管好你的API密钥。 永远不要将它粘贴到你在网上分享的文件中或公开发布。把它当作密码对待。如果你认为它已经泄露，回到Anthropic控制台删除它，然后创建一个新的。

Common Problems and How to Fix Them 常见问题及解决方法

Even the simplest installation can have hiccups. Here are the most common issues I have seen — and exactly how to fix them. 即使是最简单的安装也可能出现问题。以下是我见过的最常见问题——以及如何解决它们。

🔧 Troubleshooting Guide 🔧 故障排除指南
  • "command not found: claude" — This means Claude Code was not installed globally. Try closing the terminal completely, reopening it, and running the npm install -g command again. "command not found: claude"——这意味着Claude Code没有全局安装。尝试完全关闭终端，重新打开它，然后再次运行npm install -g命令。
  • "node: command not found" — Node.js is not installed or not set up properly. Reinstall it from nodejs.org and restart your computer before trying again. "node: command not found"——Node.js未安装或未正确设置。从nodejs.org重新安装它，并在再次尝试之前重启你的电脑。
  • "Invalid API key" — Double-check you copied the full key with no extra spaces. Keys start with sk-ant-. If still failing, generate a fresh key in the Anthropic console. "Invalid API key"——仔细检查你是否复制了完整的密钥，没有多余的空格。密钥以sk-ant-开头。如果仍然失败，在Anthropic控制台中生成一个新密钥。
  • Permission errors on Mac (EACCES) — Run the install command with sudo in front: sudo npm install -g @anthropic-ai/claude-code — then enter your Mac password when asked. Mac上的权限错误（EACCES）——在安装命令前加上sudo：sudo npm install -g @anthropic-ai/claude-code——然后在询问时输入你的Mac密码。
