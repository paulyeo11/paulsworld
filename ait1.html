🧰 AIT1: AI Create PBI Dashboard

How I used AI to build a Power BI dashboard from scratch — no coding background needed.

📖 About This Project

As a retired 70-year-old learning AI, I challenged myself to create a professional Power BI dashboard using only AI assistance. This article shares my journey — from zero knowledge to a working analytics dashboard.

Power BI is Microsoft's business intelligence tool that turns raw data into beautiful visual charts and reports. With AI guiding me step by step, I was able to build one without writing a single line of code manually.

Power BI Microsoft AI No-Code Data Visualisation Retirement Project

🪜 How I Did It — Step by Step

  • 1Asked AI (Claude) to explain what Power BI is and what it can do for a beginner.
  • 2Downloaded Power BI Desktop free from Microsoft's website with AI guidance.
  • 3Prepared a simple Excel data file with my personal data (travel, activities, expenses).
  • 4Imported the data into Power BI — AI explained each screen and button.
  • 5Created charts, bar graphs and KPI cards by following AI's instructions.
  • 6Published the dashboard and embedded it on this website.

💡 What I Learned

AI makes complex tools accessible to anyone, regardless of age or technical background. The key is to ask the right questions and take it one step at a time. I went from "What is Power BI?" to a working dashboard in just a few hours.

This project gave me confidence to explore more AI tools and data visualisation techniques as part of my retirement learning journey.
