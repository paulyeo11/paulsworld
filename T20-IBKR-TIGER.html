2. FX rate difference — This dashboard uses fixed rates (HKD 7.80, SGD 1.35). The Tiger app uses live interbank rates which change constantly.
3. Different API calculation — The Tiger API returns P&L using a "cost-of-carry" method internally, which does not always match exactly what the Tiger app displays on screen.
