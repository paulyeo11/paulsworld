📈 View Your IBKR Positions

This tool connects to Interactive Brokers via ib_async (the official replacement for ib_insync). Run the Python script on your computer while IB Gateway is open, then load the output file here to see your full portfolio in a live dashboard.

How To Use

1
Install the library: pip install ib_async
2
Download the Python script below and place it in any folder
3
Open IB Gateway → Configure → Settings → API → Enable Socket Clients. Port: 4001
4
Run: python ibkr_positions.py — it creates positions.json
5
Click Load positions.json below to see your portfolio
