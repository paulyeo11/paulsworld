Why I Decided to Build a Website我为什么决定建一个网站

After retirement, I had stories to tell. Travel adventures from 38 countries. AI experiments that amazed even me. Lessons learned from a lifetime of curiosity. I wanted a place to share all of this — not just on WhatsApp or Facebook, but a proper website that was truly mine.

The problem? I had never built a website before. I did not know HTML, CSS, or JavaScript. I did not even have a laptop available most of the time. All I had was my iPhone — and Claude AI.

"I had no technical background, no laptop, and no coding knowledge. What I had was curiosity, determination, and Claude AI."

The YouTube Video That Inspired Me激励我的那个YouTube视频

Before I started building, I watched a YouTube tutorial that gave me the confidence to try. This video showed me that building a website was possible even for a complete beginner — and that was all I needed to get started.

After watching this video, I thought — "If I can understand this, I can build my own website with the help of Claude AI!" And that is exactly what I did.

The Biggest Challenge — Finding the Right Guidance最大的挑战——找到正确的指引

The hardest part of building this website was not the technology — it was finding the right tutorial to understand how everything fit together. I spent time searching for video tutorials that explained concepts clearly for a beginner.

Once I found the right resources and understood the basic concepts of GitHub and Vercel, everything became much clearer. The key was understanding that GitHub stores your files and Vercel publishes them to the internet — automatically!

The Tools I Used我使用的工具

📲 iPhone — my only device for most of the development
🤖 Claude AI — wrote all the HTML, CSS and JavaScript code
🐙 GitHub — stores all my website files for free
▲ Vercel — publishes my website to the internet for free
📊 Google Sheets — powers my live travel record

How I Built It — Step by Step

1
Describe what I wanted to Claude. I told Claude in plain English what kind of website I wanted — a personal site with travel stories, AI articles, and a travel record section.
2
Claude wrote all the code. I did not write a single line of HTML or CSS myself. Claude generated everything — the design, the layout, the colours, and all the functionality.
3
Upload to GitHub from iPhone. I copied the code from Claude and pasted it into GitHub using Safari on my iPhone. This was tricky at first but I learned the steps quickly.
4
Vercel deployed automatically. Every time I updated a file on GitHub, Vercel automatically published the changes to my website within 1-2 minutes. No extra steps needed!
5
Keep improving. I kept asking Claude to improve the design, add new features, and fix problems. Each day the website got better and better.

What Claude AI Did For Me

Claude AI was not just a tool — it was like having a patient, knowledgeable friend available 24 hours a day. Every time I had a problem, I described it to Claude in plain English and it solved it.

Claude helped me with everything — designing the layout, adding my photo, creating the navigation menu, building the AI Journey articles, and even fixing errors when things went wrong. Without Claude, this project would have been impossible for me.

"Claude AI was like having a brilliant web developer as my personal assistant — available any time, never impatient, always helpful."

What the Website Has Today

In just two days, I built a website with three complete sections — My Travel Stories, My AI Journey, and a Live Travel Record connected to Google Sheets. The site has my personal photo, a short introduction, a Facebook link, and articles about my AI experiments.

Every time I write a new story, I just create a new HTML file and upload it to GitHub. The website detects it automatically and adds it to the list — no editing of the main page needed!

"Keep learning — age is just a number."

— PAUL YEO, 70, SINGAPORE

My Message to Other Retirees

If you are reading this and thinking "I could never do that" — I am here to tell you that you are wrong. Two months ago, I thought the same thing. Today I have a live website that I built myself, articles that people are reading, and a new passion for technology that I never expected at 70.

The tools available today — especially AI like Claude — have made it possible for anyone, regardless of age or technical background, to create things that would have required professional developers just a few years ago.

Start small. Ask questions. Make mistakes. Keep learning. Age is just a number — and curiosity has no expiry date.
