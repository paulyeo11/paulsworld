What Is Claude Code on the Web?什么是网页版Claude Code？

Claude Code on the Web is the same powerful AI coding assistant — but it runs entirely in your internet browser. You visit a website, connect your GitHub repository, and Claude gets to work. There is nothing to install on your computer. No terminal. No Node.js. No configuration files to worry about.

It works like this: Claude Code runs on Anthropic's cloud servers, not on your machine. It can still read all your files, write new pages, edit existing ones, and push changes directly to GitHub — which then automatically updates your live website. The magic still happens. You just don't have to manage any of the plumbing.

☁️ What Claude Code on the Web Can Do
  • Read every file in your GitHub repository — just like the desktop version
  • Write and edit HTML, CSS, JavaScript and any other file type
  • Create brand new pages and add them to your site
  • Commit changes to GitHub — triggering automatic deployment on Vercel
  • Remember your project context across sessions using CLAUDE.md
  • Run entirely in your browser — no installation on your computer needed

How to Get Started — It Is Simpler Than You Think如何开始——比你想象的简单

If you want to try Claude Code on the Web yourself, here is how straightforward it is:

Go to code.claude.ai

Open your browser and visit code.claude.ai. Sign in with your Anthropic account — the same account you use for Claude chat. No new account needed.

Connect your GitHub repository

Click to connect GitHub and select the repository you want Claude to work on. Claude will be given access to read and write your files. This takes about one minute to set up the first time.

Describe what you want in plain English

Just type what you need — "Add a new article about my trip to Japan" or "Update the colours on my homepage" or "Fix the navigation menu on mobile." Claude will figure out the technical details.

Review and approve the changes

Claude will show you what it plans to do before making changes. You stay in control — approve what looks good, ask it to adjust anything that doesn't. This is your safety net.

Watch your site update automatically

Once Claude pushes the changes to GitHub, your hosting service (like Vercel) detects the update and deploys your new site automatically. Usually live within 1–2 minutes — no extra steps from you.
