The Proper Setup — Step by Step

Here is what I learned from the video about setting Claude Code up the right way. These are the steps that actually matter:

Install Node.js first

Claude Code requires Node.js version 18 or higher. This is the foundation everything runs on. Open Terminal (Mac) or Command Prompt (Windows) and check your version before doing anything else.

Install Claude Code via npm

Run npm install -g @anthropic-ai/claude-code in your terminal. The -g flag means it installs globally — available anywhere on your computer, not just one folder.

Authenticate with your Anthropic account

Type claude to launch it for the first time. It will ask you to log in with your Anthropic account. This links Claude Code to your subscription — no separate billing.

Create your CLAUDE.md file

This is the most important step the video emphasised. Inside any project folder, create a file called CLAUDE.md. Write in plain English: what the project is, your coding preferences, what tools you use, and any rules Claude should follow. Claude Code reads this every time it starts — it's your standing instructions.

Configure permissions carefully

Claude Code can be set to auto-approve actions (fully autonomous) or to ask your permission before each step. For beginners, always start with asking for permission. As you gain trust and confidence, you can loosen this — but start cautious.

Use Plan Mode first

Before letting Claude Code make any changes, use Plan Mode (Shift+Tab twice). This makes Claude think through the task and show you its plan before touching anything. Review the plan, approve it, then let it work. This habit prevents mistakes.

Add MCP Servers for extra power

Once comfortable, explore MCP Servers — these are connectors that give Claude Code access to external services like Google Calendar, Gmail, your file system, and more. The video shows this turns Claude Code from a coding tool into a full workflow automation system.
